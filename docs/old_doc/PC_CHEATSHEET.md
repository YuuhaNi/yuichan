### 基本構文

#### 変数とデータ型

```yui
# 変数（型宣言不要）
x = 42              # 整数
pi = 3.14           # 浮動小数点数
name = "Alice"      # 文字列（内部は文字コード配列）
flag = 1            # 真偽値（1=真, 0=偽）

# 配列
nums = [1, 2, 3, 4, 5]
matrix = [[1, 2], [3, 4]]
empty = []

# 配列長
length = |nums|     # 5

# 配列アクセス
first = nums[0]     # 1
```

#### 標準ライブラリのインポート

```yui
標準ライブラリを使う  # ファイル先頭に記述（算術関数を使う場合は必須）
```

### 演算（すべて関数呼び出し）

#### 算術演算

| 演算 | 関数 | 例 |
|------|------|-----|
| 加算 | `和(a, b, ...)` | `和(1, 2, 3)` → `6` |
| 減算 | `差(a, b)` | `差(10, 3)` → `7` |
| 乗算 | `積(a, b, ...)` | `積(2, 3, 4)` → `24` |
| 除算 | `商(a, b)` | `商(7, 2)` → `3` |
| 剰余 | `剰余(a, b)` | `剰余(17, 5)` → `2` |
| 絶対値 | `絶対値(x)` | `絶対値(-5)` → `5` |
| 最大値 | `最大値(a, b, ...)` | `最大値(5, 2, 8)` → `8` |
| 最小値 | `最小値(a, b, ...)` | `最小値(5, 2, 8)` → `2` |

#### 型変換

| 変換 | 関数 |
|------|------|
| 整数化 | `整数化("123")` → `123` |
| 浮動小数点化 | `少数化(5)` → `5.0` |
| 文字列化 | `文字列化(123)` → `"123"` |

#### 型判定（1=真, 0=偽）

| 判定 | 関数 |
|------|------|
| 整数判定 | `整数判定(42)` → `1` |
| 浮動小数点判定 | `少数判定(3.14)` → `1` |
| 文字列判定 | `文字列判定("hello")` → `1` |

#### ビット演算

| 演算 | 関数 |
|------|------|
| AND | `論理積(12, 10)` → `8` |
| OR | `論理和(12, 10)` → `14` |
| XOR | `排他的論理和(12, 10)` → `6` |
| NOT | `ビット反転(5)` → `-6` |
| 左シフト | `左シフト(3, 2)` → `12` |
| 右シフト | `右シフト(12, 2)` → `3` |

### 制御構造

#### 条件分岐

```yui
# 基本形
もし x が 1 ならば {
    # 処理
}

# if-else
もし x が 1 ならば {
    # 処理A
}
そうでなければ {
    # 処理B
}

# 多段階
もし x が 1 ならば {
    # 処理A
}
そうでなければ {
    もし x が 2 ならば {
        # 処理B
    }
    そうでなければ {
        # 処理C
    }
}
```

#### 比較演算子

| 比較 | 構文 |
|------|------|
| 等しい | `x が 1` |
| 等しくない | `x が 1 以外` |
| 以上 | `x が 1 以上` |
| 以下 | `x が 1 以下` |
| より大きい | `x が 1 より大きい` |
| より小さい | `x が 1 より小さい` |

#### くり返し（固定回数のみ）

```yui
# 基本形
10回、くり返す {
    # 処理
}

# インデックス付き
i = 0
10回、くり返す {
    # iを使った処理
    iを増やす
}

# 配列走査
arr = [1, 2, 3, 4, 5]
length = |arr|
i = 0
length回、くり返す {
    elem = arr[i]
    # elemを使った処理
    iを増やす
}

# break
100回、くり返す {
    もし 条件 ならば {
        くり返しを抜ける
    }
}
```

**重要**:
- ループ回数は**0以上の整数**でなければならない
- `while` や `for` は存在しない
- `continue` は存在しない

### 関数定義

#### 引数ありの関数

```yui
標準ライブラリを使う

# 1引数
double = 入力 x に対し {
    積(x, 2)が答え
}

# 複数引数
add = 入力 x, y に対し {
    和(x, y)が答え
}

# 再帰
factorial = 入力 n に対し {
    もし n が 1 以下 ならば {
        1が答え
    }
    そうでなければ {
        prev = 差(n, 1)
        積(n, factorial(prev))が答え
    }
}
```

#### 引数なしの関数

```yui
get_answer = 入力なしに対し {
    42が答え
}
```

**重要**:
- `値が答え` で値を返す（複数箇所に書ける）
- ブロック `{` の後は**必ず改行**

### 配列操作

#### 基本操作

```yui
# 作成
arr = []
arr = [1, 2, 3]

# アクセス
first = arr[0]
arr[0] = 10

# 長さ
length = |arr|

# 末尾に追加
arrの末尾に 4 を 追加する

# インクリメント・デクリメント
arr[0]を増やす  # arr[0] += 1
arr[0]を減らす  # arr[0] -= 1
```

### 文字列操作

**重要**: 文字列は**文字コード（整数）の配列**

#### 基本操作

```yui
# 文字列リテラル
text = "Hello"      # 内部: [72, 101, 108, 108, 111]

# 長さ
length = |text|     # 5

# 文字コード取得
code = text[0]      # 72 ('H')

# 文字コードから文字列
char_code = "A"[0]  # 65

# 文字列の変更（ミュータブル）
text[0] = "Z"[0]    # "Zello"

# 末尾に文字を追加
textの末尾に 33 を 追加する  # "Hello!" (33 = '!')
```

#### 文字列の比較

```yui
標準ライブラリを使う

# 文字列は直接比較できない（文字ごとに比較）
str1 = "ABC"
str2 = "ABC"
len1 = |str1|
len2 = |str2|

is_equal = 1
もし len1 が len2 以外 ならば {
    is_equal = 0
}
そうでなければ {
    i = 0
    len1回、くり返す {
        もし str1[i] が str2[i] 以外 ならば {
            is_equal = 0
            くり返しを抜ける
        }
        iを増やす
    }
}
```
