# Yui言語リファレンス - LLM向けコンテキスト

## 概要

Yui（ゆい）は日本語でプログラムを書ける教育用プログラミング言語です。
生成AI時代に対応した、自然な日本語構文を持つチューリング完全な言語です。

**設計思想:**

- 日本語の自然な文法に従う
- 最小限の操作セットでチューリング完全を実現
- 変数名は英語でも日本語でもOK（コード例では英語を推奨）

## データ型

```yui
user_name = "Alice"          # 文字列（内部は文字コードの配列）
age = 25                     # 整数
height = 1.65                # 小数 固定精度
is_student = 1               # 論理値: 真 (1) か 偽 (0)
alpha = "A"[0]               # 65, 文字コード

nums = [1, 2, 3]             # 配列
fruits = ["apple", "banana", "orange"]  # 文字列配列
matrix = [
    [1, 2],
    [3, 4]
]
```

## 基本構文

### 1. 変数と演算

```yui
x = 10        # 代入
xを増やす     # x += 1 インクリメント
xを減らす     # x -= 1 デクリメント

arr = [0, 1, 2]
arr[0]を増やす  # arr[0] += 1 インクリメント
arr[0]を減らす  # arr[0] -= 1 デクリメント
```

注意：Yuiに四則演算子はない（標準ライブラリを使う）

### 2. 条件分岐

```yui
もし x が 1 ならば {
    # 真の場合の処理
}

もし x が 1 ならば {
    # 真の場合
}
そうでなければ {
    # 偽の場合
}
```

注意：「そうでなければ」は省略可能

**比較演算子:**

```yui
もし x が 1 ならば { }                  # x == 1 等しい
もし x が 1 以外 ならば { }              # x != 1 等しくない
もし x が 1 以上 ならば { }              # x >= 1 以上
もし x が 1 以下 ならば { }              # x <= 1 以下
もし x が 1 より大きい ならば { }         # x > 1 より大きい
もし x が 1 より小さい ならば { }         # x < 1 より小さい
```

**ネスト例:**

```yui
score = 85
もし score が 90 以上 ならば {
    grade = "A"
}
そうでなければ {
    もし score が 80 以上 ならば {
        grade = "B"
    }
    そうでなければ {
        もし score が 60 以上 ならば {
            grade = "C"
        }
        そうでなければ {
            grade = "F"
        }
    }
}
```

**重要**: `{` のあとは必ず改行が必要

### 3. くり返し

あらかじめ十分なくり返し回数を予測する

```yui
x = 0
10回、くり返す {
    xを増やす
    もし x が 5 ならば {
        くり返しを抜ける  # break
    }
}
```

### 4. 配列操作

末尾に追加できる

```yui
array = [1, 2, 3]

length = |array|  # 配列の長さ

# 末尾に追加
arrayの末尾に 4 を 追加する        # [1, 2, 3, 4]
```

### 5. 文字列操作

文字列は文字コードの配列、変更可能、末尾に追加できる

```yui
string = "ABC"

length = |string|  # 文字列の長さ (3)

# 末尾に追加
stringの末尾に 65 を 追加する        # "ABCA"

# 文字列の変更
string[0] = "Z"[0]  # "ZBCA"
```

### 6. 関数定義

Yuiは四則演算を含め、簡単な計算はすべて関数を定義する。

```yui
function_name = 入力 x, y に対し {
    # 処理
    result が答え    # return
}
```

**引数なしの関数**は、`入力なしに対し` と定義します。

```yui
function_name = 入力なしに対し {
    # 処理
    result が答え    # return
}
```

**例: 引数なしの関数**

```yui
標準ライブラリを使う

get_answer = 入力なしに対し {
    和(40, 2)が答え   # return 42
}

result = get_answer()    # 42
```

**例: 2倍にする関数**

```yui
標準ライブラリを使う

double = 入力 x に対し {
    積(x, 2)が答え   # return x * 2
}

result = double(5)    # 10
```

### 7. テストケース

Yuiは、トップレベルにdoctestの形式で書くと、テストになる

```yui
>>> double(5)
10
```

### 1. 標準ライブラリの読み込み

### 2. 標準ライブラリ関数

標準ライブラリ関数は**可変長引数**に対応しています。

#### 算術関数（可変長引数、小数も対応）

```yui
標準ライブラリを使う

q = 商(7, 2)                       # 3 (整数同士は切り捨て)
q = 商(7.0, 2.0)                   # 3.5 (小数除算)
r = 剰余(17, 5)                    # 2

# 配列変数を渡すことも可能
nums = [1, 2, 3, 4, 5]
sum = 和(nums)                      # 15
```

#### 比較・統計関数

```yui
標準ライブラリを使う

max_val = 最大値(5, 2, 8, 1)       # 8 (可変長引数)
min_val = 最小値(5, 2, 8, 1)       # 1 (可変長引数)
abs_val = 絶対値(-5)               # 5
```

#### 型変換・判定関数

```yui
標準ライブラリを使う

# 型変換
n = 整数化("123")            # 123
n = 整数化(1.7)              # 1
f = 少数化(5)                # 5.0
f = 少数化("3.14")           # 3.14

# 型判定（戻り値: 1=True, 0=False）

is_int = 整数判定(3.14)      # 0
is_float = 少数判定(3.14)    # 1
is_float = 少数判定(42)      # 0
is_str = 文字列判定("hello") # 1
is_str = 文字列判定(123)     # 0
```

#### ビット演算関数

```yui
標準ライブラリを使う

# ビット演算（32ビット符号なし整数）
and_val = 論理積(12, 10)              # 8  (1100 & 1010 = 1000)
or_val = 論理和(12, 10)               # 14 (1100 | 1010 = 1110)
xor_val = 排他的論理和(12, 10)        # 6  (1100 ^ 1010 = 0110)
not_val = ビット反転(5)               # -6 (ビット反転)

# シフト演算
left = 左シフト(3, 2)                 # 12 (0011 << 2 = 1100)
right = 右シフト(12, 2)               # 3  (1100 >> 2 = 0011)

# 32ビットに制限する場合
val = 剰余(left, 4294967296)          # 32ビット符号なし整数
```

#### その他の関数

```yui

```

### 例8: バブルソート

```yui
標準ライブラリを使う

bubble_sort = 入力 arr に対し {
    n = |arr|

    i = 0
    n回、くり返す {
        j = 0
        limit = 差(差(n, i), 1)

        limit回、くり返す {
            j_next = 和(j, 1)

            もし arr[j] が arr[j_next] より大きい ならば {
                # スワップ
                temp = arr[j]
                arr[j] = arr[j_next]
                arr[j_next] = temp
            }

            jを増やす
        }
        iを増やす
    }

    arrが答え
}

>>> bubble_sort([5, 2, 8, 1, 9])
[1, 2, 5, 8, 9]
```

### 例9: MD5ハッシュ（高度な例）

```yui
標準ライブラリを使う

# MD5のメイン処理（簡略版）
md5_round = 入力 A, B, C, D, M, s, K に対し {
    # F = (B AND C) OR ((NOT B) AND D)
    not_B = 剰余(ビット反転(B), 4294967296)
    F = 論理和(論理積(B, C), 論理積(not_B, D))

    # F = F + A + K + M
    F = 和(F, A)
    F = 和(F, K)
    F = 和(F, M)
    F = 剰余(F, 4294967296)

    # Left rotate
    left_part = 剰余(左シフト(F, s), 4294967296)
    right_part = 右シフト(F, 差(32, s))
    F = 剰余(論理和(left_part, right_part), 4294967296)

    # F = F + B
    F = 和(F, B)
    F = 剰余(F, 4294967296)

    Fが答え
}
```

## 重要な注意事項

### 1. 標準ライブラリの読み込み

算術関数（和、差、積、商など）を使う前に、必ず以下を記述：

```yui
標準ライブラリを使う
```

### 2. 関数呼び出しには必ず括弧

```yui
# ❌ 間違い
result = 和

# ✅ 正しい
result = 和(1, 2, 3)
```

### 3. 可変長引数と配列

標準ライブラリ関数は可変長引数に対応しています：

```yui
標準ライブラリを使う

# 可変長引数で渡す（推奨）
total = 和(1, 2, 3, 4, 5)
diff = 差(10, 3)
prod = 積(2, 3, 4)

# 配列変数を渡すことも可能
nums = [1, 2, 3, 4, 5]
total = 和(nums)
```

### 4. 戻り値の指定

関数から値を返すには `値が答え` を使います：

```yui
function_name = 入力 x に対し {
    resultが答え    # これが戻り値
}
```

### 5. 変数名は英語でも日本語でもOK

```yui
# 両方とも有効
x = 10
value = 10

# 日本語も可能（ただしコード例では英語推奨）
値 = 10
```

### 6. ブロックの後は必ず改行

```yui
# ❌ 間違い
もし x が 1 ならば { xを増やす }

# ✅ 正しい
もし x が 1 ならば {
    xを増やす
}
```

### 7. 文字列は文字コードの配列

```yui
s = "ABC"
# 内部的には [65, 66, 67]

first_char = s[0]  # 65
length = |s|       # 3

# 文字コードから文字列を構築
char_a = "A"[0]    # 65
```

### 8. 浮動小数点演算は明示的に

```yui
標準ライブラリを使う

# 整数除算（切り捨て）
result = 商(7, 2)  # 3

# 浮動小数点除算
result = 商(7.0, 2.0)  # 3.5
# または
result = 商(少数化(7), 少数化(2))  # 3.5
```

### 文字列の解析

```yui
標準ライブラリを使う

# カンマを小数点として扱う
parse_number = 入力 str に対し {
    length = |str|
    int_part = 0
    frac_part = 0.0
    found_decimal = 0
    decimal_places = 0

    i = 0
    length回、くり返す {
        char = str[i]

        # 数字かチェック (48-57)
        is_digit = 0
        digit = 0
        もし char が 47 より大きい ならば {
            もし char が 58 より小さい ならば {
                is_digit = 1
                digit = 差(char, 48)
            }
        }

        もし is_digit が 1 ならば {
            もし found_decimal が 0 ならば {
                int_part = 和(積(int_part, 10), digit)
            }
            そうでなければ {
                decimal_placesを増やす
                divisor = 1.0
                j = 0
                decimal_places回、くり返す {
                    divisor = 積(divisor, 10.0)
                    jを増やす
                }
                digit_f = 少数化(digit)
                frac = 商(digit_f, divisor)
                frac_part = 和(frac_part, frac)
            }
        }

        # カンマ(44)または小数点(46)
        もし char が 44 ならば {
            found_decimal = 1
        }
        もし char が 46 ならば {
            found_decimal = 1
        }

        iを増やす
    }

    int_f = 少数化(int_part)
    result = 和(int_f, frac_part)
    resultが答え
}

>>> parse_number("2,5")
2.5
>>> parse_number("3.14")
3.14
```

## コード生成時のチェックリスト

LLMがYuiコードを生成する際の確認事項：

- [ ] 標準ライブラリ関数を使う場合、`標準ライブラリを使う` を記述したか
- [ ] 関数呼び出しに括弧 `()` を付けたか
- [ ] 標準ライブラリ関数に可変長引数を渡しているか（`和(1, 2, 3)` 推奨）
- [ ] 配列変数を渡す場合も正しく記述しているか（`和(配列変数)`）
- [ ] 条件文の構文が正しいか（`もし X ならば { }`）
- [ ] ブロックの `{` の後に改行があるか
- [ ] 繰り返しの構文が正しいか（`N回、くり返す { }`）
- [ ] 関数定義の構文が正しいか（`入力 X に対し { Y が答え }`）
- [ ] 変数の増減に `を増やす` / `を減らす` を使っているか
- [ ] 配列操作の構文が正しいか（`の末尾に X を 追加する`）
- [ ] 変数名は英語を使っているか（推奨）
- [ ] 浮動小数点演算で明示的に `少数化()` を使っているか

## まとめ

Yui言語は、日本語の自然な文法を活かした教育用プログラミング言語です。
この文書の例を参考に、LLMはYuiのコードを生成できます。

**基本パターンを組み合わせることで、複雑なアルゴリズムも実装可能です。**

### 標準ライブラリの全機能

- **算術**: 和, 差, 積, 商, 剰余
- **統計**: 最大値, 最小値, 絶対値
- **型変換**: 整数化, 少数化, 文字列化
- **型判定**: 整数判定, 少数判定, 文字列判定
- **ビット演算**: 論理積, 論理和, 排他的論理和, ビット反転
- **シフト演算**: 左シフト, 右シフト
- **その他**: 乱数生成

これらを組み合わせることで、MD5ハッシュのような複雑な暗号化アルゴリズムも実装可能です。
