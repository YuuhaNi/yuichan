## Yui言語 実行時エラーメッセージ集

Yuiインタプリタが出力する全エラーメッセージと、その原因・発生例・修正例を示します。

---

### 1. 変数エラー

#### 「知らない変数だよ！ 'x'」

原因: 定義されていない変数を参照した

[発生例]
```yui
y = x
```

[修正例]
```yui
x = 0
y = x
```

---

### 2. 配列エラー

#### 2-1. 「配列ではありません: 123」

原因: 配列でない値に添え字アクセスした

[発生例]
```yui
x = 123
y = x[0]
```

[修正例]
```yui
x = [1, 2, 3]
y = x[0]
```

---

#### 2-2. 「配列の添え字は0から2の間ですよ: 5」

原因: 配列の範囲外にアクセスした

[発生例]
```yui

x = arr[5]
```

[修正例]
```yui
arr = [1, 2, 3]
x = arr[2]
```

---

#### 2-3. 「配列は空です。添え字の確認も: 0」

原因: 空の配列に添え字アクセスした

[発生例]
```yui

x = arr[0]
```

[修正例]
```yui
arr = [1]
x = arr[0]
```

---

#### 2-4. 「配列じゃないね？」

原因: 配列でない値に「の末尾に〜を追加する」を使用した

[発生例]
```yui

xの末尾に 4 を 追加する
```

[修正例]
```yui
x = [1, 2, 3]
xの末尾に 4 を 追加する
```

---

#### 2-5. 「配列じゃないよ」

原因: 配列でない値に対して要素の検索を行おうとした

[発生例]
```yui

```

[修正例]
```yui

```

### 6. 実行制限エラー

#### 6-1. 「再帰呼び出しが深さが上限(=512)を超えました」

原因: 再帰が深すぎる、または無限再帰

[発生例]
```yui
f = 入力 x に対し {
    f(x)が答え
}

>>> f(1)
```

[修正例]
```yui
標準ライブラリを使う

f = 入力 x に対し {
    もし x が 0 以下 ならば {
        0が答え
    }
    そうでなければ {
        prev = 差(x, 1)
        和(x, f(prev))が答え
    }
}

>>> f(5)
15
```

### 7. テストエラー

#### 7-1. 「テストに失敗: 5 (正解) 10」

原因: 関数の出力が期待値と一致しない

[発生例]
```yui
標準ライブラリを使う

double = 入力 x に対し {
    xが答え
}

>>> double(5)
10
```

[修正例]
```yui
標準ライブラリを使う

double = 入力 x に対し {
    積(x, 2)が答え
}

>>> double(5)
10
```
