## Yui言語の説明

2025年12月

Yui言語は、日本語ベースの教育用プログラミング言語です。
この文書では、Yui言語の基本を説明します。

### 1 変数と値

変数名は、英字で始まる英数字と `_` の並びです。

**例**: `kosu`, `kosu_gokei`, `tokuten`

**推奨**: 変数名は英語で記述します。

配列の要素は、要素の番号を添字で指定します。2次元以上の場合は、添字を `,` で区切ります。たとえば、(1次元の) 配列 `tokuten` や 2次元配列 `gyoretu` の要素は `tokuten[2]` や `gyoretu[3][2]` のように表します。

添字の値は 0 以上の整数です。

特に断らない限り、数値は 10進法で表します。文字列は、文字の並びを `"` と `"` でくくって表します。

**重要**: Yui言語では、**文字列は文字コード（整数）の配列**として扱われます。

**例**: `100`

**例**: `99.999`

**例**: `"Hello"`（内部的には `[72, 101, 108, 108, 111]` として扱われます）


### 2 代入文

代入文は変数に値を設定します。`=` の左辺に変数または添字付きの配列を、右辺に代入する値を書きます。

**例**: `kosu = 3`

**例**: `tokuten[4] = 100`

**例**: 配列の各要素に同じ値を代入（手動でループを使用）

```yui
標準ライブラリを使う

init_array = 入力 size に対し {
    tokuten = []
    i = 0
    size回、くり返す {
        tokutenの末尾に 0 を 追加する
        iを増やす
    }
    tokutenが答え
}
```

**例**: 配列リテラルによる初期化
```yui
tokuten = [87, 45, 72, 100]
```

**例**: `kosu を増やす` は `kosu = 和(kosu, 1)` と同じです。

**例**: `kosu を減らす` は `kosu = 差(kosu, 1)` と同じです。

### 3 演算

この節では、算術演算と比較演算、そして論理演算について説明します。比較演算やそれを組み合わせる論理演算は、条件分岐文（5.1節）や繰返し文（5.2節）の〈条件〉で使うことができます。

#### 4.1 算術演算

Yui言語では、**算術演算は標準ライブラリ関数**として提供されます。ファイルの先頭に `標準ライブラリを使う` を記述する必要があります。

**DNCL との対応**:

| DNCL | Yui | 説明 |
|------|-----|------|
| `a ＋ b` | `和(a, b)` | 加算 |
| `a － b` | `差(a, b)` | 減算 |
| `a × b` | `積(a, b)` | 乗算 |
| `a / b` | `商(a, b)` | 整数除算（商） |
| `a ％ b` | `剰余(a, b)` | 余り |

**例**:
```yui
標準ライブラリを使う

atai = 商(7, 2)  # atai には 3 が代入される（整数除算）
```

**例**:
```yui
標準ライブラリを使う

syo = 商(7, 2)       # 3
amari = 剰余(10, 3)  # 1
```

複数の演算を組み合わせる場合は、関数呼び出しをネストします。

**例**:
```yui
標準ライブラリを使う

sogaku = 差(差(1, 2), 3)
```

**例**:
```yui
標準ライブラリを使う

kazu = 10
kosu = 和(1, 商(kazu, 3))
```

**例**:
```yui
標準ライブラリを使う

hidari = 1
migi = 3
heikin = 商(和(hidari, migi), 2)

>>> heikin
2
```

#### 4.2 比較演算

数値の比較演算は、以下の形式で記述します。演算結果は、真（1）か偽（0）の値となります。

**DNCL との対応**:

| DNCL | Yui |
|------|-----|
| `kosu ＝ 3` | `kosu が 3` |
| `kosu ≠ 3` | `kosu が 3 以外` |
| `kosu ＞ 3` | `kosu が 3 より大きい` |
| `kosu ≧ 3` | `kosu が 3 以上` |
| `kosu ≦ 3` | `kosu が 3 以下` |
| `kosu ＜ 3` | `kosu が 3 より小さい` |

**例**:
```yui
kosu = 4
もし kosu が 3 より大きいならば、{
    # 処理
}
```

**例**:
```yui
標準ライブラリを使う

ninzu = 5
もし 積(ninzu, 2) が 8 以下 ならば、{
    #処理
}
そうでなければ {
    #処理
}
```

**例**:
```yui
kaisu = 5
もし kaisu が 0 以外ならば、{
    # 処理
}
```

文字列の比較は、文字コード配列として文字ごとに比較する必要があります。

**注意**: Yui言語では、文字列は文字コード配列なので、直接比較はできません。

**例**: 文字列の等価性チェック
```yui
標準ライブラリを使う

string_equals = 入力 str1, str2 に対し {
    len1 = |str1|
    len2 = |str2|

    もし len1 が len2 以外ならば、{
        0が答え
    }

    i = 0
    len1回、くり返す {
        もし str1[i] が str2[i] 以外ならば、{
            0が答え
        }
        iを増やす
    }

    1が答え
}

>>> string_equals("ABC", "ABC")
1
>>> string_equals("ABC", "abc")
0
```

#### 4.3 論理演算

Yui言語には、論理演算子がありません。条件式を組み合わせる場合は、ネストした条件分岐を使用します。

**例**:

```yui
kosu = 15
もし kosu が 12 以上 ならば、{
    もし kosu が 27 以下 ならば、{
        # kosu が 12以上 27以下の場合の処理
    }
}
```

**例**:

```yui
標準ライブラリを使う

check = 入力 kosu に対し {
    remainder = 剰余(kosu, 2)
    もし remainder が 0 ならば、{
        1が答え  # 偶数
    }
    もし kosu が 0 より小さいならば、{
        1が答え  # 負
    }
    0が答え
}
```

**例**:
```yui
kosu = 100
もし kosu が 75 以下 ならば、{
    # kosu が 75 より大きくない場合の処理
}
```

### 5 制御文

条件分岐文（5.1節）と繰返し文（5.2節）をまとめて制御文と呼びます。

#### 5.1 条件分岐文

条件分岐文は、〈条件〉が成り立つかどうかによって、実行する処理を切り替えます。

**重要**: Yui言語では、すべての `{` の後に**必ず改行**が必要です。

##### 基本形（条件が真の場合のみ処理）

**例**:
```yui
標準ライブラリを使う

process = 入力 x, y に対し {
    もし x が 3 より小さいならば、{
        x を増やす
        y を減らす
    }
    [x, y]が答え
}

>>> process(2, 5)
[3, 4]
>>> process(5, 5)
[5, 5]
```

##### if-else形（条件に応じて異なる処理）

**例**:
```yui
標準ライブラリを使う

process = 入力 x に対し {
    もし x が 3 より小さいならば、{
        x を増やす
    }
    そうでなければ {
        x を減らす
    }
    xが答え
}

>>> process(2)
3
>>> process(5)
4
```

##### 多段階条件分岐

**例**:
```yui
標準ライブラリを使う

process = 入力 x, y に対し {
    もし x が 3 ならば、{
        x を増やす
    }
    そうでなければ {
        もし y が 2 より大きいならば、{
            y を増やす
        }
        そうでなければ {
            y を減らす
        }
    }
    [x, y]が答え
}

>>> process(3, 5)
[4, 5]
>>> process(1, 5)
[1, 6]
>>> process(1, 1)
[1, 0]
```

#### 5.2 くり返し文

Yui言語のくり返し文は、**固定回数のくり返し**のみをサポートします。

**例**:（固定回数で代用）
```yui
標準ライブラリを使う

sum_while = 入力 x に対し {
    gokei = 0
    max_iterations = 100  # 十分大きな回数
    i = 0

    max_iterations回、くり返す {
        もし x が 10 以上 ならば、{
            くり返しを抜ける
        }
        gokei = 和(gokei, x)
        xを増やす
        iを増やす
    }

    gokeiが答え
}

>>> sum_while(5)
35
```

**例**:
```yui
標準ライブラリを使う

sum_reverse = 入力 gokei に対し {
    x = 10

    10回、くり返す {
        gokei = 和(gokei, x)
        xを減らす
    }

    gokeiが答え
}

>>> sum_reverse(0)
55
```

##### ループ制御

Yui言語は `くり返しを抜ける`（break）をサポートします。

**例**: break の使用
```yui
標準ライブラリを使う

find_value = 入力 array, target に対し {
    length = |array|
    result = -1
    i = 0

    length回、くり返す {
        もし array[i] が target ならば、{
            result = i
            くり返しを抜ける
        }
        iを増やす
    }

    resultが答え
}

>>> find_value([1, 2, 3, 4, 5], 3)
2
```

## 6 標準ライブラリ関数

Yui言語では、算術演算やその他の処理を標準ライブラリ関数として提供します。これらの関数を使用するには、ファイルの先頭に `標準ライブラリを使う` を記述する必要があります。

### 算術関数

| 関数 | 説明 | DNCL 対応 |
|------|------|-----------|
| `和(a, b, c, ...)` | 加算 | `a ＋ b ＋ c` |
| `差(a, b)` | 減算 | `a － b` |
| `積(a, b, c, ...)` | 乗算 | `a × b × c` |
| `商(a, b)` | 整数除算 | `a / b` または `a ÷ b` |
| `剰余(a, b)` | 剰余 | `a ％ b` |
| `絶対値(x)` | 絶対値 | - |
| `最大値(a, b, c, ...)` | 最大値 | - |
| `最小値(a, b, c, ...)` | 最小値 | - |

### 型変換・判定関数

| 関数 | 説明 |
|------|------|
| `整数化(x)` | 整数に変換 |
| `少数化(x)` | 浮動小数点数に変換 |
| `文字列化(x)` | 文字列（文字コード配列）に変換 |
| `整数判定(x)` | 整数なら 1、そうでなければ 0 |
| `少数判定(x)` | 浮動小数点数なら 1、そうでなければ 0 |
| `文字列判定(x)` | 文字列なら 1、そうでなければ 0 |
| `乱数生成()` | 0.0~1.0の乱数を生成する |

### ビット演算関数

| 関数 | 説明 |
|------|------|
| `論理積(x, y)` | ビット AND |
| `論理和(x, y)` | ビット OR |
| `排他的論理和(x, y)` | ビット XOR |
| `ビット反転(x)` | ビット NOT |
| `左シフト(x, n)` | 左シフト (x << n) |
| `右シフト(x, n)` | 右シフト (x >> n) |
