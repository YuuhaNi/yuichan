## Yui Language Survival Guide for C Programmers

**5分でYuiを理解する** - C言語プログラマーのための最速ガイド

### TL;DR (超クイックスタート)

```c
// C
int factorial(int n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

printf("%d\n", factorial(5));  // 120
```

```yui
# Yui
標準ライブラリを使う

factorial = 入力 n に対し {
    もし n が 1 以下 ならば {
        1が答え
    }
    そうでなければ {
        prev = 差(n, 1)
        積(n, factorial(prev))が答え
    }
}

>>> factorial(5)
120
```

**重要な違い**:
- 型宣言不要（動的型付け）
- 四則演算は関数呼び出し: `a + b` → `和(a, b)`
- ポインタなし、すべて値渡し
- ブロックは `{` で始まり、**必ず改行**が必要
- テストケースは `>>>` で書く（printf不要）

### データ型の対比

### 基本型

| C | Yui | 備考 |
|---|-----|------|
| `int x = 42;` | `x = 42` | 整数（型宣言不要） |
| `float x = 3.14;` | `x = 3.14` | 浮動小数点数 |
| `double x = 3.14;` | `x = 3.14` | Yuiでは区別なし |
| `bool x = true;` | `x = 1` | 真偽値は 1/0 |
| `bool x = false;` | `x = 0` | 真偽値は 1/0 |
| `char* str = "Hello";` | `str = "Hello"` | 文字列（内部は文字コード配列） |
| `char c = 'A';` | `c = "A"[0]` | 文字コード（65） |

### 配列

```c
// C
int nums[5] = {1, 2, 3, 4, 5};
int matrix[2][2] = {{1, 2}, {3, 4}};
char* text = "ABC";
```

```yui
# Yui
nums = [1, 2, 3, 4, 5]      # 動的配列
matrix = [[1, 2], [3, 4]]   # 多次元配列
text = "ABC"                 # 内部は [65, 66, 67]
```

**重要な違い**:

| C | Yui |
|---|-----|
| 固定サイズ配列 | 動的配列（自動拡張） |
| `int arr[10];` | `arr = []` |
| `sizeof(arr) / sizeof(int)` | `\|arr\|` |
| null終端文字列 | 長さ情報を持つ配列 |
| ポインタ演算可能 | ポインタなし |

## 制御構造の対比

### 条件分岐

```c
// C
if (x == 1) {
    printf("one\n");
} else if (x == 2) {
    printf("two\n");
} else {
    printf("other\n");
}
```

```yui
# Yui
もし x が 1 ならば {
    # one
}
そうでなければ {
    もし x が 2 ならば {
        # two
    }
    そうでなければ {
        # other
    }
}
```

### 比較演算子

| C | Yui |
|---|-----|
| `x == 1` | `x が 1` |
| `x != 1` | `x が 1 以外` |
| `x >= 1` | `x が 1 以上` |
| `x <= 1` | `x が 1 以下` |
| `x > 1` | `x が 1 より大きい` |
| `x < 1` | `x が 1 より小さい` |

### ループ

```c
// C: for loop
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}
```

```yui
# Yui: 固定回数ループ
i = 0
10回、くり返す {
    # iを使った処理
    iを増やす
}
```

```c
// C: while loop
while (x < 10) {
    x++;
}
```

```yui
# Yui: 十分大きい回数でループ
1000回、くり返す {
    もし x が 10 より小さい ならば {
        xを増やす
    }
}
```

**重要**: Yuiには `while` も `for(;;)` もない。固定回数のループのみ。

### Break文

```c
// C
for (int i = 0; i < 100; i++) {
    if (i == 5) {
        break;
    }
}
```

```yui
# Yui
i = 0
100回、くり返す {
    もし i が 5 ならば {
        くり返しを抜ける
    }
    iを増やす
}
```

**注意**: Yuiには `continue` がありません。

## 関数定義の対比

```c
// C
int add(int x, int y) {
    return x + y;
}

int factorial(int n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

int get_answer(void) {
    return 42;
}
```

```yui
# Yui
標準ライブラリを使う

add = 入力 x, y に対し {
    和(x, y)が答え
}

factorial = 入力 n に対し {
    もし n が 1 以下 ならば {
        1が答え
    }
    そうでなければ {
        prev = 差(n, 1)
        積(n, factorial(prev))が答え
    }
}

get_answer = 入力なしに対し {
    42が答え
}
```

**Yuiの関数定義パターン**:
```yui
# 引数ありの関数
関数名 = 入力 引数1, 引数2 に対し {
    # 処理
    戻り値 が答え
}

# 引数なしの関数
関数名 = 入力なしに対し {
    # 処理
    戻り値 が答え
}
```

**重要な違い**:
- 型宣言不要（動的型付け）
- `return` の代わりに `値が答え`
- `void` パラメータの代わりに `入力なしに対し`

## 演算子の変換表

### 算術演算

| C | Yui | 備考 |
|---|-----|------|
| `a + b` | `和(a, b)` | 可変長引数: `和(a, b, c)` |
| `a - b` | `差(a, b)` | |
| `a * b` | `積(a, b)` | 可変長引数: `積(a, b, c)` |
| `a / b` | `商(a, b)` | int÷int=int（切り捨て） |
| `a % b` | `剰余(a, b)` | |
| `abs(a)` | `絶対値(a)` | |
| `fmax(a, b)` | `最大値(a, b)` | 可変長引数 |
| `fmin(a, b)` | `最小値(a, b)` | 可変長引数 |

### 型変換

| C | Yui |
|---|-----|
| `(int)x` | `整数化(x)` |
| `(float)x` | `少数化(x)` |
| `sprintf(buf, "%d", x)` | `文字列化(x)` |

### ビット演算

| C | Yui |
|---|-----|
| `a & b` | `論理積(a, b)` |
| `a \| b` | `論理和(a, b)` |
| `a ^ b` | `排他的論理和(a, b)` |
| `~a` | `ビット反転(a)` |
| `a << b` | `左シフト(a, b)` |
| `a >> b` | `右シフト(a, b)` |

### インクリメント・デクリメント

| C | Yui |
|---|-----|
| `x++` | `xを増やす` |
| `x--` | `xを減らす` |
| `arr[i]++` | `arr[i]を増やす` |
| `++x` | `xを増やす`（前置・後置の区別なし） |


## 標準ライブラリ

### 必須のインポート

```yui
標準ライブラリを使う
```

算術関数を使う前に**必ず**記述！（C言語の `#include` に相当）

### 主要な関数

```yui
標準ライブラリを使う

# 算術
sum = 和(1, 2, 3, 4, 5)        # 15
diff = 差(10, 3)                # 7
prod = 積(2, 3, 4)              # 24

# 比較・統計
max_val = 最大値(5, 2, 8, 1)    # 8
min_val = 最小値(5, 2, 8, 1)    # 1

# 型変換

n = 整数化(1.7)                 # 1

s = 文字列化(123)               # "123"

# 型判定
is_int = 整数判定(42)           # 1 (true)
is_float = 少数判定(3.14)       # 1 (true)

# ビット演算
```

## 落とし穴と回避策

### 1. 演算子を使わない

```c
// C: NG (Yuiでは使えない)
int x = a + b;
```

```yui
# Yui: OK
標準ライブラリを使う
x = 和(a, b)
```

### 2. ブロックの後は必ず改行

```c
// C: OK
if (x == 1) { x++; }
```

```yui
# Yui: NG
もし x が 1 ならば { xを増やす }

# Yui: OK
もし x が 1 ならば {
    xを増やす
}
```

### 3. whileループは存在しない

```c
// C: OK
while (x < 10) {
    x++;
}
```

```yui
# Yui: 十分大きい回数でループ
1000回、くり返す {
    もし x が 10 より小さい ならば {
        xを増やす
    }
}
```

### 4. ポインタ渡しは不可

```c
// C: ポインタ渡し
void increment(int* x) {
    (*x)++;
}

int val = 5;
increment(&val);  // val は 6
```

```yui
# Yui: 値を返す
標準ライブラリを使う

increment = 入力 x に対し {
    和(x, 1)が答え
}

val = 5
val = increment(val)  # val は 6
```

### 5. 配列の境界チェック

```c
// C: 境界チェックなし（バッファオーバーラン）
int arr[5] = {1, 2, 3, 4, 5};
int x = arr[10];  // 未定義動作
```

```yui
# Yui: 事前にチェック
arr = [1, 2, 3, 4, 5]
index = 10
len = |arr|
もし index が len より小さい ならば {
    x = arr[index]
}
そうでなければ {
    # エラー処理
}
```

### 6. NULL終端文字列ではない

```c
// C: NULL終端
char str[] = "Hello";  // "Hello\0"
int len = strlen(str); // 5
```

```yui
# Yui: 長さ情報を持つ配列
str = "Hello"  # [72, 101, 108, 108, 111]
len = |str|    # 5（NULL文字なし）
```
