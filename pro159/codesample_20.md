```yui
標準ライブラリを使う
# 簡易スタックマシン
# 命令は [opcode, operand] の形の配列で表す。operand が不要な命令は 0 を入れる。
#
# 対応命令:
#   ["PUSH", n] -- スタックに n を積む
#   ["POP",  0] -- スタック頂上を捨てる
#   ["DUP",  0] -- スタック頂上を複製する
#   ["ADD",  0] -- 上 2 つを取り出して a+b を積む
#   ["SUB",  0] -- 上 2 つを取り出して a-b を積む
#   ["MUL",  0] -- 上 2 つを取り出して a*b を積む
#
# プログラム実行後、スタックの頂上を答えとして返す。
# (Yui の配列は要素削除ができないので、論理的なスタックの大きさを top で管理し、
#  pop は top を減らすだけにする)

run=入力programに対し{
   stack=[]
   top=0
   n=programの大きさ
   i=0
   n回くり返す{
      instr=program[i]
      op=instr[0]
      arg=instr[1]

      もしopが"PUSH"ならば{
         # 論理 top が配列実長より小さければ上書き、そうでなければ末尾に追加
         sz=stackの大きさ
         もしtopがszより小さいならば{
            stack[top]=arg
         }
         そうでなければ{
            stackにargを追加する
         }
         topを増やす
      }
      もしopが"POP"ならば{
         topを減らす
      }
      もしopが"DUP"ならば{
         t=差(top,1)
         x=stack[t]
         sz=stackの大きさ
         もしtopがszより小さいならば{
            stack[top]=x
         }
         そうでなければ{
            stackにxを追加する
         }
         topを増やす
      }
      もしopが"ADD"ならば{
         topを減らす
         b=stack[top]
         topを減らす
         a=stack[top]
         stack[top]=和(a,b)
         topを増やす
      }
      もしopが"SUB"ならば{
         topを減らす
         b=stack[top]
         topを減らす
         a=stack[top]
         stack[top]=差(a,b)
         topを増やす
      }
      もしopが"MUL"ならば{
         topを減らす
         b=stack[top]
         topを減らす
         a=stack[top]
         stack[top]=積(a,b)
         topを増やす
      }
      iを増やす
   }
   # スタックの頂上を結果として返す
   topを減らす
   stack[top]が答え
}

# テスト: 3 + 4 = 7
>>> run([["PUSH",3],["PUSH",4],["ADD",0]])
7
# テスト: (3 + 4) * 2 = 14
>>> run([["PUSH",3],["PUSH",4],["ADD",0],["PUSH",2],["MUL",0]])
14
# テスト: 10 - (2 + 3) = 5
>>> run([["PUSH",10],["PUSH",2],["PUSH",3],["ADD",0],["SUB",0]])
5
# テスト: DUP で 5 を複製して掛ける → 25
>>> run([["PUSH",5],["DUP",0],["MUL",0]])
25
# テスト: POP で途中の値を捨てる → 7
>>> run([["PUSH",7],["PUSH",99],["POP",0]])
7
# テスト: 単一 PUSH
>>> run([["PUSH",42]])
42
```

```yui
# 二次方程式 ax^2 + bx + c = 0 の実数解を求める (実数解 2 つを想定)
# 解の公式:  x = (-b ± √D) / 2a    ただし D = b^2 - 4ac
# 結果は [大きい解, 小さい解] の配列で返す

solve=入力a,b,cに対し{
   D=差(積(b,b),積(4,積(a,c)))
   sqrtD=平方根(D)
   # 整数同士の 商() は床除算になるので、すべて小数化してから割る
   denom=小数化(積(2,a))
   mb=小数化(差(0,b))
   x1=商(和(mb,sqrtD),denom)
   x2=商(差(mb,sqrtD),denom)
   [x1,x2]が答え
}

# テスト: x^2 - 3x + 2 = 0 → x = 2, 1
```

```yui
標準ライブラリを使う
# 文字列 s の中で部分文字列 sub が最初に現れる位置 (0 始まり) を返す
# 見つからなければ -1 を返す
# 文字列は文字コード配列なので、配列としても同じように使える

find=入力s,subに対し{
   n=sの大きさ
   m=subの大きさ
   neg1=差(0,1)
   pos=neg1
   # 空 sub は常に位置 0 で一致
   もしmが0ならば{
      pos=0
   }
   そうでなければ{
      もしnがmより小さいならば{
         何もしない
      }
      そうでなければ{
         # 走査範囲は 0..n-m
         limit=和(差(n,m),1)
         i=0
         limit回くり返す{
            # 既に見つかっていたらスキップ (まだ -1 のときだけ調べる)
            もしposがneg1ならば{
               match=1
               j=0
               m回くり返す{
                  ij=和(i,j)
                  cs=s[ij]
                  ct=sub[j]
                  もしcsがct以外ならば{
                     match=0
                  }
                  jを増やす
               }
               もしmatchが1ならば{
                  pos=i
               }
            }
            iを増やす
         }
      }
   }
   posが答え
}

>>> find("hello world","world")

# テスト: 先頭で一致
>>> find("hello","hello")

>>> find("hello","xyz")

>>> find("hello","")

# テスト: sub が s より長い
>>> find("hi","hello")

# テスト: 末尾近くで一致
```
