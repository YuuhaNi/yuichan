# Yui 標準ライブラリ API 仕様書（ユーザ向け）

`標準ライブラリを使う` と宣言すると、本書に挙げる関数群が利用できるようになります。

```yui
標準ライブラリを使う
>>> 和(1, 2, 3)
6
```

---

## 1. 数学関数

### 絶対値

数値の絶対値を返します。

- `絶対値(x)` → 数値（`x` が整数なら整数、小数なら小数）
- `x` は整数または小数。

```yui
>>> 絶対値(-5)
5
```

### 平方根

非負数の平方根を **小数** で返します。負の数を渡すとエラー。

- `平方根(x)` → 小数

```yui
>>> 平方根(9)
3.000000
```

### 乱数

0 以上 1 未満のランダムな小数を返します。引数なし。

- `乱数()` → 小数

---

## 2. 算術関数

四則演算は演算子ではなく、以下の関数として提供されます。
**可変長引数**を受け取り、引数を 1 つの配列で渡してもよい（`和([1,2,3])` と `和(1,2,3)` は同じ）。
引数のうち 1 つでも小数があれば結果は小数、すべて整数なら結果も整数になります。

### 和

要素の合計。

- `和(x, y, …)` または `和(配列)` → 数値

```yui
>>> 和(1, 2, 3)
6
>>> 和([1, 2, 3])
6
```

### 差

先頭の値から残りを順に引いた結果。

- `差(x, y, …)` → 数値

```yui
>>> 差(10, 3, 2)
5
```

### 積

要素を順に掛け合わせた結果。

- `積(x, y, …)` → 数値

### 商

先頭の値を残りの値で順に割った結果。整数同士は **床除算**（切り捨て）。
0 で割るとエラー。

- `商(x, y, …)` → 数値

```yui
>>> 商(10, 3)
3
>>> 商(10.0, 3.0)
3.333333
```

### 剰余

割り算の余りを返します。0 で割るとエラー。

- `剰余(x, y, …)` → 数値

```yui
>>> 剰余(10, 3)
1
```

### 最大値

引数の中の最大値を返します。

- `最大値(x, y, …)` → 数値

### 最小値

引数の中の最小値を返します。

- `最小値(x, y, …)` → 数値

## 3. 型判定関数

引数の型を調べ、論理値（`真` / `偽`）を返します。

| 関数 | 真になる条件 |
|------|--------------|
| `ブール判定(x)` | `x` が論理値 |
| `整数判定(x)` | `x` が整数 |
| `小数判定(x)` | `x` が小数 |
| `文字列判定(x)` | `x` が文字列 |
| `配列判定(x)` | `x` が配列 |
| `オブジェクト判定(x)` | `x` がオブジェクト |

```yui
>>> 整数判定(42)
真
>>> 文字列判定([1,2,3])
偽
```

---

## 4. 型変換関数

値を別の型へ変換して返します。

### 整数化

値を整数に変換します。`値なし` は 0、配列は文字コード列として整数化を試みます。
変換できない場合はエラー。

- `整数化(x)` → 整数

```yui
>>> 整数化("42")
42
>>> 整数化(3.7)
3
```

### 小数化

値を小数に変換します。`値なし` は `0.0`。

- `小数化(x)` → 小数

### 文字列化

値を文字列に変換します。小数は小数点以下 6 桁の固定書式（例: `"3.140000"`）になります。

- `文字列化(x)` → 文字列

```yui
>>> 文字列化(42)
"42"
>>> 文字列化(3.14)
"3.140000"
```

### 配列化

値を配列に変換します。文字列は文字コードの配列として、オブジェクトはキー一覧として返されます。

- `配列化(x)` → 配列

```yui
>>> 配列化("Hi")
[72, 105]
>>> 配列化({"x":1, "y":2})
["x", "y"]
```

---

## 5. エラーについて

標準ライブラリ関数は次の場合にエラーを発生させます。

| エラー | 発生条件 |
|--------|----------|
| `mismatch-argument` | 引数の個数が合わない |
| `not-negative-number` | `平方根` に負の数を渡した |
| `division-by-zero` | `商` または `剰余` で 0 で割った |
| `int-conversion` / `float-conversion` | `整数化` / `小数化` が変換に失敗 |

エラーは実行時に停止し、`❌` / `✅` 付きのメッセージで原因を示します。
